14. Se citesc 2 valori in ordine crescatoare. Sa se afiseze toate numerele dintre cele 2 valori citite. Daca nu exista valori, de afisat, se afiseaza un mesaj de notificare.

Exemple:

Se citeste -2 si 4. Se afiseaza -1 0 1 2 3

Se citeste 3 si 4. Se afiseaza mesajul “Nu exista nici o valoare de afisat”

Se citeste 4 si 4. Se afiseaza mesajul “Nu exista nici o valoare de afisat”

Rezultatul va fi afisat aici

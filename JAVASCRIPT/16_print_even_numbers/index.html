16. Se citesc 2 valori in ordine crescatoare. Sa se afiseze toate numerele pare care sunt mai mari sau egale cu primul numar si mai mici sau egale cu al doilea numar.

Exemple:

Se citeste -3 si 3. Se afiseaza -2 0 2

Se citeste 3 si 4. Se afiseaza 4

Se citeste 3 si 3. Se afiseaza “Nici o valoare de afisat”

Rezultatul va fi afisat aici
